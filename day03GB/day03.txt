배열
	같은 자료형의 여러 변수들을 나열해 선언해 놓은 것

배열의 선언
	자료형[ ] 배열명 = {값1, 값2, 값3, ...}
	자료형[ ] 배열명 = new 자료형[칸수];

배열의 사용
	배열명[방번호] = 값;
	arData[2] = 20;

배열의 사용 이유
	1. 여러 비슷한 의미의 값들을 하나의 이름과 방번호(인덱스)로 관리하기 위해서
	2. 규칙성이 없는 여러 값들에 규칙성을 부여하기 위해서	

배열의 인덱스 번호
	실제 값들이 저장될 수 있는 공간이 할당되면 이름은 존재하지 않고 주소만 존재한다.
	따라서 각 공간들에 쉽게 접근할 수 있도록 첫번째 시작 주소를 배열명 변수가
	기억하고 있고, 주소연산(추가이동)을 통해 다음 주소로 이동한다.
	시작 주소에서 연산된 정수(이동한 횟수)가 해당 방의 인덱스 번호가 되고, 첫번째 방은
	배열명 변수가 가지고 있는 시작 주소에서 0번 추가 이동해야 하므로 인덱스 번호는
	0번부터 시작한다.

배열의 길이
	배열명.length
